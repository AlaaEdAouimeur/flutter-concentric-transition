## Getting Started
* Add this to your pubspec.yaml
  ```
  dependencies:
  concentric_transition: ^0.0.1
  ```
* Get the package from Pub:
  ```
  flutter packages get
  ```
* Import it in your file
  ```
  import 'package:concentric_transition/concentric_transition.dart';
  ```
